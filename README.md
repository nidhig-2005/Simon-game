Simon Game – Memory Challenge

A fun, colourful Simon Game built using HTML, CSS, and JavaScript.
Test your memory, follow the pattern, and see how far you can go!

Features:
Interactive & colourful UI
Random pattern generation
Increasing difficulty with each round
Fully responsive for all devices

Technologies Used:
HTML
CSS
JavaScript (DOM + Events)

How to Play:
click Start button to begin the game.
Watch the sequence of blinking colours.
Repeat the pattern by clicking the buttons in order.
Each level adds one more colour to the sequence.
If you press the wrong button — Game Over!
